Dodge Ram TRX Fanpage

Це простий веб-сайт, присвячений пікапу **Dodge Ram TRX**, з інформацією про машину, її історію, технічні характеристики, особливості та галерею.

Використані технології

- HTML5
- CSS
- JavaScript
- Секції сайту:
  - Головна
  - Про TRX
  - Історія Dodge Ram TRX
  - Технічні характеристики
  - Галерея
  - Контакти

Особливості сайту

- Лого збільшене для кращого вигляду
- Hero фото відображає повністю автомобіль
- Галерея у два ряди з більшими фото
- Підписи до фотографій білі і видимі на будь-якому фоні
- Плавна прокрутка по сторінці при натисканні на меню
- Адаптивний дизайн для мобільних пристроїв

Як запустити

1. Скачайте або клоніть проект.
2. Відкрийте `index.html` у браузері.
3. Насолоджуйтесь сайтом!
